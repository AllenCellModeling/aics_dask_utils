Installation¶

Stable release¶

To install AICS Dask Utils, run this command in your terminal:

$ pip install aics_dask_utils

This is the preferred method to install AICS Dask Utils, as it will always install the most recent stable release.

If you don’t have pip installed, this Python installation guide can guide you through the process.

From sources¶

The sources for AICS Dask Utils can be downloaded from the Github repo.

You can either clone the public repository:

$ git clone git://github.com/AllenCellModeling/aics_dask_utils

Or download the tarball:

$ curl  -OL https://github.com/AllenCellModeling/aics_dask_utils/tarball/master

Once you have a copy of the source, you can install it with:

$ python setup.py install
